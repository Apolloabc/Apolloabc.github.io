使用Hugo创建个人博客 June 26, 2019 Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。

Table of Contents 第一步 安装Hugo 第二步 创建新的站点 第三步 下载主题 第四步 生成文章 第一步 安装Hugo 在windows环境下，首先下载Scoop包管理工具，利用Scoop安装Hugo

scoop install hugo 第二步 创建新的站点 hugo new site myblog（自定义名字） 第三步 下载主题 Hugo主题下载

打开powershell，进入新建博客的themes目录，git clone相应的主题

hugo server -t 你的theme名称 –buildDrafts 有的主题需要将其exampleSite文件夹中的config.toml文件复制到根目录中才能完整预览效果 第四步 生成文章 myblog/content是用来存放文档的地方，在其下建立一个新的md文件(放在post文件夹中，post文件夹可以自己手动新建)

hugo new post/first.md 在first.md中写入一些markdown格式的内容，使用如下命令进行本地运行

hugo server -t (下载的主题名字) –buildDrafts 在http://localhost:1313/中查看效果
