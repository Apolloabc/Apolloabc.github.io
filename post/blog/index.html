  新建并上传hugo博客

  July 19, 2019

scoop包管理工具下载安装包时默认从国外网下载，体验较差，还是自己从国内镜像下载吧（跟maven设置镜像应该是一样的原理）。

  • 第一步：安装git

  • 第二步：安装hugo

  • 第三步：新建站点 E:\Blog\Sites> hugo new site my-blog

  • 第四步：下载主题 E:\Blog\Sites\my-blog>git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

  • 第五步：本机部署 E:\Blog\Sites\my-blog>hugo server -t m10c --buildDrafts

  访问本机部署的博客 http://localhost:1313/

  • 第六步：新建博客 E:\Blog\Sites\my-blog>hugo new post/blog.md 编辑博客（Typora）

  • 第七步：重复第五步

  • 第八步①：新建github仓库，在本机准备站点（生成public文件夹）

  E:\Blog\Sites\my-blog>hugo --theme=m10c --baseUrl="https://Apolloabc.github.io/" --buildDrafts

  • 第八步②：git操作

  E:\Blog\Sites\my-blog\public>git init

  E:\Blog\Sites\my-blog\public>git add .
  E:\Blog\Sites\my-blog\public>git commit -m “我的hugo博客，第一次提交”
  E:\Blog\Sites\my-blog\public>git remote add origin
  E:\Blog\Sites\my-blog\public>git push -u origin master

  • 第九步：通过公网访问发布在github中的博客 https://apolloabc.github.io/
