新建并上传hugo博客\n July 2, 2019\n

scoop包管理工具下载安装包时默认从国外网下载，体验较差，还是自己从国内镜像下载吧（跟maven设置镜像应该是一样的原理）。\n

第一步：安装git http://npm.taobao.org/mirrors/git-for-windows/

第二步：安装hugo https://gohugo.io/getting-started/installing/

第三步：新建站点 E:\Blog\Sites> hugo new site my-blog

第四步：下载主题 E:\Blog\Sites\my-blog>git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

第五步：本机部署 E:\Blog\Sites\my-blog>hugo server -t m10c –buildDrafts http://localhost:1313/

第六步：新建博客 E:\Blog\Sites\my-blog>hugo new post/blog.md 编辑博客（VS Code）

第七步：重复第五步

第八步：新建github仓库，在本机准备站点（生成public文件夹）\n
E:\Blog\Sites\my-blog>hugo –theme=m10c –baseUrl=“https://Apolloabc.github.io/" –buildDrafts

第九步：git操作 E:\Blog\Sites\my-blog\public>git init\n E:\Blog\Sites\my-blog\public>git add .\n E:\Blog\Sites\my-blog\public>git commit -m “我的hugo博客，第一次提交”\n E:\Blog\Sites\my-blog\public>git remote add origin https://github.com/Apolloabc/Apolloabc.github.io.git \n E:\Blog\Sites\my-blog\public>git push -u origin master\n https://apolloabc.github.io/
